# IBSGlobal

Welcome to **IBSGlobal** — a Minecraft 1.20.1 server built on a massive Earth map with the power of the Create Mod!

Explore real-world continents, build industrial cities, create automated factories, and form nations with other players. Whether you want to become an engineer, trader, explorer, or ruler, IBSGlobal offers a world full of possibilities.

## Features

* 🌍 Realistic Earth Map
* ⚙️ Create Mod Automation & Engineering
* 🏙️ Player-built Cities and Nations
* 🚂 Transportation Systems & Infrastructure
* 🤝 Community Economy and Trading
* 🛠️ Minecraft Java Edition 1.20.1

## Server Vision

IBSGlobal combines creativity, industry, and geopolitics in a persistent Earth-based survival experience. Build massive machines, connect continents, and shape the future of the world together.

## Version

**Minecraft Java 1.20.1**
